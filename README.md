# Hệ thống Phân tích Cầu lông bằng AI (Badminton AI Coach)

Đồ án Tốt nghiệp - [Năm học]
Sinh viên: [Tên của bạn] - [MSSV]

## 1. Giới thiệu
Dự án sử dụng Computer Vision (YOLOv8) để phân tích kỹ thuật và chiến thuật trong môn cầu lông. Hệ thống có khả năng nhận diện dáng người, vẽ Minimap sân đấu và thống kê chỉ số trận đấu.

## 2. Cấu trúc thư mục
- **SourceCode/**: Chứa toàn bộ mã nguồn chương trình.
- **Documents/**: Chứa báo cáo tiến độ, báo cáo giữa kỳ và cuối kỳ.

## 3. Hướng dẫn cài đặt
1. Clone dự án:
   `git clone <link-repo-cua-ban>`
2. Cài đặt thư viện:
   `cd SourceCode`
   `pip install -r requirements.txt`
3. Chạy chương trình:
   `python main.py`
